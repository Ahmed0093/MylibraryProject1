# MylibraryProject1

This project is aimed for learning and to get hands dirty to publish a library online.
and make A Custom View in Android .

## Getting Started
The library contains
make a custom view in a library and puplish it with 'jitpack.io’ 
adding expandable bottom navigation bar. 



### TO DO

## HOW TO USE

* In your project’s build.gradle add the following line

    maven { url ‘https://jitpack.io’ }

* and in your app’s build.gradle add the dependency

  implementation 'com.github.Ahmed0093:mylibraryproject1:-SNAPSHOT'
